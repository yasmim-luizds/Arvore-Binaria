# Árvore Binária de Pesquisa em Java

Projeto desenvolvido em **Java** com o objetivo de aplicar o conceito de **Árvore Binária de Pesquisa** em um sistema simples de gerenciamento de produtos de mercado.

---

## Sobre o Projeto

Este projeto simula um sistema de mercado, onde os produtos são armazenados em uma **árvore binária de pesquisa**.

Cada produto possui informações como nome, código de barras, preço, estoque e categoria. A organização da árvore é feita com base no nome do produto, permitindo realizar operações como cadastro, busca, remoção e atualização.

O sistema é executado pelo terminal e utiliza um menu interativo para que o usuário escolha as ações desejadas.

---

## Funcionalidades

- Login de acesso ao sistema
- Cadastro de produtos
- Consulta de produtos por nome
- Remoção de produtos
- Atualização de preço, estoque e categoria
- Contagem de produtos por categoria
- Listagem de produtos por faixa de preço
- Exibição do produto mais barato
- Exibição do produto mais caro

---

## Tecnologias Utilizadas

- Java
- Programação Orientada a Objetos
- Árvore Binária de Pesquisa
- Entrada de dados pelo terminal
